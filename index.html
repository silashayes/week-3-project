Project 1: Data Analytics
The first project I worked on simulated a task I'd be assigned as a data science intern in an analytics / visualization-focused role. I was assigned a dataset, was given a specific question through which to analyze the data, and presented my data-driven answers to this question through a slide deck presentation and interactive data visualization.

Project 2: Database Structures
The second project was designed to give me practice designing and pulling data from a wide variety of database architectures. Collaboratively, my group and I created a set of data based on ourselves, used it to populate three different database architectures, and wrote scripts to query each of them.

Project 3: Intro to Software Engineering
This project prepared me to more smoothly integrate into Scenthound's more software engineering-focused tasks. I learned the basics of HTML, CSS, and JavaScript, and then hosted a basic website on Netlify. Additionally, I learned about how React can link up with Firestore by creating an interactive app using that software stack.
